# Merchant Dashboard

A modern, responsive merchant management dashboard built with React, TypeScript, and Vite.

## Features

- Dashboard with key metrics and interactive charts
- Complete merchant management (Create, Read, Update, Delete)
- Advanced filtering by status and risk level
- Real-time search and sorting
- Infinite scroll pagination
- Light and dark theme toggle
- Fully responsive design
- Form validation with business rules

## Technologies Used

- React 18 with TypeScript
- Vite (Build tool)
- React Router DOM (Routing)
- Zustand (State management)
- React Hook Form + Zod (Form validation)
- Recharts (Data visualization)
- CSS Modules (Styling)
- Lucide React (Icons)
- React Intersection Observer (Infinite scroll)


## How to Run the Project

### Prerequisites
- Node.js (v16 or higher)

### Steps

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```
